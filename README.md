つくったもの／調べたことの概要
・RPGバトルシステム
　TypeScriptを用い、クラスの継承とオーバーライドを学びながら作成

つまずいた点
　ゲッターとセッターを実装する際、変数名とメソッド名を同じ（hp）にしてしまい無限ループのエラーを発生させたが、内部データ用の変数に _（アンダースコア）をつける慣習を知り解決できた。
　

工夫した点
　魔法使いの攻撃に Math.random() と Math.floor() を組み合わせ、ランダムに威力が変わる要素を追加した。

学んだこと
　・セッターの具体的な使い方
　・親クラスの型（Character[]）で配列を作ることで、Warrior や Mage などの異なる子クラスをひとまとめに格納し、ループ処理で一気に動かせる便利さを実践的に学んだ。
